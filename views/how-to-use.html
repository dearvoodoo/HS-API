How to Use
The API ?
The API returns a JSON response by default.
You can use query in url to request a specific information.
All list of endpoints and result exemple are available in the page Endpoints.
Note: API requests may be rate limited.
How to get image ?
API can return a path for an image or an icon.
You have to build the path yourself. But no worries this simple as fuck.


All images are uploaded to my website.
So take this part https://huntshowdown.dearvoodoo.com/imgs
and just add the path that the API returns.

So an image link looks like this:
https://huntshowdown.dearvoodoo.com/imgs/ammo/compact/normal_ammo_box.png
How to use queries ?
Queries are very usefull when you want specific information to be return.
For example, you want information on the "Bolt Thrower" trait.
You can use the ID of this trait to directly tell at the API that you want only information with this tag.
ex: https://huntshowdown-api.herokuapp.com/trait_info?tag=bolt_thrower
You can request more than one information at a time.
Rate Limit
This API has a Rate-Limit.
Each IP can request up to 100 requests per 15 minute window.

You can see in the headers of your request the information related to your limitation.


If you reach this limit this error 429 will be return.
Global eferences ?
You are probably asking yourself the question "Ok this is good but what exactly are the numbers?"

So the answer is this:
effective_range is calculated in meters. (m)
rate_of_fire is calculated in revolutions per minute. (rpm)
handling is calculated in percentages. (%)
reload_speed is calculated in seconds. (s)
muzzle_velocity is calculated in meters per second. (m/s)
cost is calculated in Hunt Dollars. (HD)
price can be:
    • BB for Blood Bonds
    • HD for Hunt Dollars
    • DLC for the DLC ID (steam) build like this: https://store.steampowered.com/app/ID_HERE
    • TRIAL for trial rewards level
    • EVENT for the event name
    • TW for Twitch Drops
